# Copilot Chat Modes — Templates

This folder contains templates and instructions for adding Copilot Chat modes or using the prompts in the Copilot Chat window.

Usage
- Copy the JSON files into GitHub Copilot Chat (if your Copilot version supports importing custom modes) or copy the `instructions` text into a new Copilot Chat conversation.
- Each mode includes recommended system and user instructions to guide the assistant toward the role (e.g., "Blog Writer", "Researcher").

Files
- `blog_writer_mode.json` — Blog creation mode (intake → scan → outline → draft → edit)
- `researcher_mode.json` — Focused web-research and references
- `editor_mode.json` — Edit pass with Anti-AI rules

If your Copilot Chat UI does not support importing, copy/paste the `instructions` field contents into a new conversation and pin/save it where possible.
