# Rubber Duck: The Last Junior Developer

## Discovery Elements

1. **The claim:** AI is collapsing the career ladder from the bottom. By automating the "grunt work," it removes the low-stakes failure environment where junior developers build the judgment necessary to become senior engineers. Without this pipeline, we are setting up a long-term talent desert.
2. **The audience:** Engineering leaders, senior developers, and tech executives.
3. **The narrative anchor:** The traditional 3 onboarding scenarios (The Codebase Crawl, the PoC Environment Setup, and Pair Programming).
4. **The scope (what gets cut):** Will not focus purely on AI feature capabilities, but tightly on the *workflow friction* required to build cognitive maps.
